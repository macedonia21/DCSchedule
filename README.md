## A Meteor 1.8, React 16, React Router 5, Bootstrap 4

DC Resource Schedule project

## Quick start
Clone repository:
```
git clone https://github.com/johnwils/meteor-react-template.git
```
Install packages:
```
meteor npm install
```
Start Meteor:
```
meteor
```

Navigate to [http://localhost:3000](http://localhost:3000) in any browser.
